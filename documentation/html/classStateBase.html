Detailed Description

Defines an interface through which Game class interacts with current state.

Implementation of "state" design pattern. The functions handleEvent(), update(), render(), etc can be called based on what state the game is in. In the future, states like Main Menu, Practice mode, Level editor, etc. can be easily added by inheriting from this class.
