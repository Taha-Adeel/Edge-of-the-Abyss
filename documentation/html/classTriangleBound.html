bool TriangleBound::checkCollision ( const BoxBound &  playerBound,
const TriangleBound &  spike 
)
static

Check collision between a boxbound(the player) and a triangle bounds.

First does a "broad phase collision detection", and if it is triggered, then does a "narrow phase collision detection". This is done as the narrow phase is resource intensive while broad phase detection is very fast.
