Detailed Description

template<typename Resource>
class AssetPool< Resource >

Template class that keeps track of assets of typename Resource.

AssetPool only loads a resource from a file if it hadn't loaded it in before. As processing from a file takes time and consumes more memory it is an advantage.

Template Parameters
ResourceThe type of resource that our AssetPool holds. Valid for sf::Texture,sf::Font, st::SoundBuffer.
